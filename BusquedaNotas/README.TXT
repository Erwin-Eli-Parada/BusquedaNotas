Para este programa la idea fue buscar las notas en las cuales aparece una palabra que dada por el usuario y para ello se creó un metodo
que compara tanto el titulo de la nota como su cuerpo para saber si contiene esa palabra,además se creó un método que lee un archivo 
de acuerdo a determinado nombre y regresará toda la información del archivo en forma de un string y las notas separadas por por comas,
y por último está el método main para probar como seria la implemetación de los dos anteriores.
